## How the Payment Gateway Works (Using Actix-web + SeaORM)

<img src="https://github.com/MamangRust/payment-gateway-actix/raw/main/payment_gateway.png?raw=true" alt="payment" />

### Migrate

```sh
sea-orm-cli migrate init
```

### Generate Entity

```sh
sea-orm-cli generate entity -o src/entities
```

### Migrate run UP

```sh
sea-orm-cli migrate up
```

### Migrate run Down

```sh
sea-orm-cli migrate down
```
