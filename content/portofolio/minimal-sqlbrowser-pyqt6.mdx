# 🗃️ Minimal SQLite Database App

This application is a **simple example** of using **PyQt6** to interact with an **SQLite database**. It allows you to **create, open, and save** SQLite databases, as well as perform basic operations like **reading, writing, deleting, and updating** tables.

## ✨ Key Features
- **Create Database** 🏗️: Generate a new SQLite database with default or user-defined tables.
- **Open Database** 🔓: Select and open an existing SQLite database.
- **Save Database** 💾: Save the currently opened database.
- **Table Operations** 🧑‍💻: View, add, update, and delete rows in a table.
- **Search Function** 🔍: Search for data in tables based on specific criteria.
- **Export SQL** 📤: Export table data to an SQL file.

## 🛠 Technologies Used
- **Python** 🐍: The main programming language.
- **PyQt6** 🖥️: GUI framework for building desktop applications with Python.
- **SQLite** 🗄️: Lightweight, file-based database for data storage.
