# 💳 Monolithic Payment Gateway

This project implements a **Monolithic Payment Gateway**, designed to handle various financial operations using modern technologies to ensure scalability, observability, and seamless service integration.

## 🚀 Technologies Used

✅ **FastAPI** 🚀 - A high-performance Python-based API framework with asynchronous support.

✅ **Kafka & Zookeeper** 🐘🐍 - Used as a message broker for reliable asynchronous communication.

✅ **OpenTelemetry & Jaeger** 🐆🌐 - Tools for observability, monitoring, and distributed tracing.

✅ **Grafana & Prometheus** 📊⚙️ - For real-time monitoring and system performance analysis.

## 🔥 Key Features

- **Centralized Financial Management**: Authentication, balance tracking, top-up, transfer, withdrawal, and email notifications.
- **Comprehensive Observability**: Integrated with OTel, Jaeger, Prometheus, and Grafana.
- **Scalable Communication**: Utilizes Kafka for efficient transaction processing.
- **Modular Design**: Well-structured service architecture for easy development and maintenance.
