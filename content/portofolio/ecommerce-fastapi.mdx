# 🛒 Ecommerce FastAPI React

This project is an **e-commerce platform** developed using **FastAPI** for the backend and **React** for the frontend. FastAPI was chosen due to its high performance and its syntax similarity to Express.js.

## ✨ Features

- **FastAPI-based API** 🚀 – Provides a high-performance backend.
- **Frontend with React and Tailwind CSS** 🖥️🌊 – Modern UI with a responsive design.
- **JWT Authentication** 🔐 – Secure user authentication system.

## 🛠 Technologies Used

- **FastAPI** 🚀 – High-performance web framework for building APIs.
- **React.js** ⚛️ – Modern frontend library for dynamic UI.
- **PostgreSQL** 🗃️ – Relational database management system.
