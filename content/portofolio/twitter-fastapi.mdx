# Twitter Clone FastAPI

This project is a clone of Twitter built using FastAPI. The framework was chosen due to its similarity to Express.js, as well as its speed and efficiency in building REST APIs.

## Features

- CRUD for posts
- JWT Authentication

## Technologies Used

- FastAPI
- PostgreSQL
