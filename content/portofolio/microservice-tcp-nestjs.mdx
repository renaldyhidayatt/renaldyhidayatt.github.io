# 🛒 MyExperiment with nx.js and NestJS Backend

This project is an **experiment** in building an **e-commerce platform** using modern technologies. It integrates **Midtrans** as a payment gateway and **Raja Ongkir** for shipping services. The frontend is designed for **user convenience** and **management**, utilizing **React.js** and **Tauri**.

## ✨ Key Features

- **Backend** 💻: Built with **Nest.js** and **PostgreSQL** 🗃️, storing transaction and product data, while integrating **Midtrans** 💳 for payments and **Raja Ongkir** 📦 for shipping.
- **Frontend** 🖥️: Developed using **React.js** ⚛️ and **Redux Toolkit** 🔄, with **Tailwind CSS** 🌊 and **Flowbite** 💡 for a responsive UI.
- **Admin Panel** 🛠️: A desktop admin application built with **Tauri** 🦀 and **Rust** 🦠, featuring **React.js** ⚛️, **Redux Toolkit** 🔄, **Bootstrap** ⚙️, and **Mazer Dashboard** 📊 for an intuitive management interface.

## 🛠 Technologies Used

- **Backend** 💻: TypeScript, **Nest.js** 🏰, **PostgreSQL** 🗃️
- **Payment Gateway** 💳: **Midtrans**
- **Shipping Service** 📦: **Raja Ongkir**
- **Frontend** 🖥️: JavaScript, **React.js** ⚛️, **Redux Toolkit** 🔄, **Tailwind CSS** 🌊, **Flowbite** 💡
- **Admin Panel** 🛠️: **Tauri** 🦀, **Rust** 🦠, **React.js** ⚛️, **Redux Toolkit** 🔄, **Bootstrap** ⚙️, **Mazer Dashboard** 📊
