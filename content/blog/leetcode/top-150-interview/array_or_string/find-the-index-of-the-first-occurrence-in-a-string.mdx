## Problem Statement

Given two strings `needle` and `haystack`, return the index of the first occurrence of `needle` in `haystack`, or `-1` if `needle` is not part of `haystack`.

### Example:

**Input:**
```
haystack = "sadbutsad", needle = "sad"
```
**Output:**
```
0
```

## Approach

1. Use the built-in substring search function if available.
2. If not, iterate over `haystack` and check for `needle` manually.

## Algorithm

1. Loop through `haystack` from index `0` to `len(haystack) - len(needle)`.
2. Check if substring `haystack[i:i+len(needle)]` matches `needle`.
3. If found, return `i`.
4. If not found, return `-1`.

## Implementations

### Implementation in Go

```go
package main

import (
    "fmt"
    "strings"
)

func strStr(haystack string, needle string) int {
    return strings.Index(haystack, needle)
}

func main() {
    fmt.Println(strStr("sadbutsad", "sad")) // Output: 0
}
```

### Implementation in Java

```java
class Solution {
    public int strStr(String haystack, String needle) {
        return haystack.indexOf(needle);
    }
    
    public static void main(String[] args) {
        Solution sol = new Solution();
        System.out.println(sol.strStr("sadbutsad", "sad")); // Output: 0
    }
}
```

## Explanation

- **Using built-in functions:** Directly finds `needle` in `haystack`.
- **Manual iteration:** If functions are unavailable, loop and compare substrings.

## Time and Space Complexity

- **Time Complexity:** `O(n*m)` for manual checking, `O(n)` for built-in functions.
- **Space Complexity:** `O(1)` since no extra space is used.
