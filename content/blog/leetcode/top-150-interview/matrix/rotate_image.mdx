## Problem Statement

You are given an `n x n` 2D matrix representing an image, rotate the image by 90 degrees (clockwise).

You have to rotate the image **in-place**, which means you have to modify the input 2D matrix directly. **DO NOT** allocate another 2D matrix and do the rotation.

### Example:

**Input:**
```
matrix = [[1,2,3],[4,5,6],[7,8,9]]
```
**Output:**
```
[[7,4,1],[8,5,2],[9,6,3]]
```

## Approach

1. **Transpose the matrix** by swapping `matrix[i][j]` with `matrix[j][i]`.
2. **Reverse each row** to achieve the 90-degree rotation.

## Algorithm

1. Iterate over the matrix to swap elements across the diagonal (transpose).
2. Reverse each row to complete the rotation.

## Implementations

### Implementation in Go

```go
package main

import "fmt"

func rotate(matrix [][]int) {
    n := len(matrix)
    for i := 0; i < n; i++ {
        for j := i + 1; j < n; j++ {
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        }
    }
    
    for i := 0; i < n; i++ {
        for j, k := 0, n-1; j < k; j, k = j+1, k-1 {
            matrix[i][j], matrix[i][k] = matrix[i][k], matrix[i][j]
        }
    }
}

func main() {
    matrix := [][]int{{1, 2, 3}, {4, 5, 6}, {7, 8, 9}}
    rotate(matrix)
    fmt.Println(matrix)
}
```

### Implementation in Java

```java
import java.util.*;

class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
        
        for (int i = 0; i < n; i++) {
            for (int j = 0, k = n - 1; j < k; j++, k--) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[i][k];
                matrix[i][k] = temp;
            }
        }
    }
}
```

## Explanation

- **Step 1:** Transpose the matrix by swapping elements along the diagonal.
- **Step 2:** Reverse each row to achieve a 90-degree clockwise rotation.

## Time and Space Complexity

- **Time Complexity:** `O(n^2)`, since we iterate over the entire matrix twice.
- **Space Complexity:** `O(1)`, as the transformation is done in-place.
