## Problem Statement
Given a string s and an integer k, return the maximum number of vowel letters in any substring of s with length k.

Vowel letters in English are 'a', 'e', 'i', 'o', and 'u'.

### Example 1:

**Input:** s = "abciiidef", k = 3  
**Output:** 3  
**Explanation:** The substring "iii" contains 3 vowel letters.

### Example 2:

**Input:** s = "aeiou", k = 2  
**Output:** 2  
**Explanation:** Any substring of length 2 contains 2 vowels.

### Example 3:

**Input:** s = "leetcode", k = 3  
**Output:** 2  
**Explanation:** "lee", "eet" and "ode" contain 2 vowels.

### Constraints:

- 1 <= s.length <= 10^5
- s consists of lowercase English letters.
- 1 <= k <= s.length

## Approach

We use the sliding window technique to efficiently find the maximum number of vowels in any substring of length k.

## Algorithm

1. Compute the number of vowels in the first k characters.
2. Slide the window across the string, updating the count by subtracting the leftmost character and adding the next character.
3. Keep track of the maximum vowel count found.
4. Return the maximum vowel count.

## Implementations

### Implementation in Go

```go
func maxVowels(s string, k int) int {
    vowels := map[byte]bool{'a': true, 'e': true, 'i': true, 'o': true, 'u': true}
    count := 0
    for i := 0; i < k; i++ {
        if vowels[s[i]] {
            count++
        }
    }
    maxCount := count
    for i := k; i < len(s); i++ {
        if vowels[s[i]] {
            count++
        }
        if vowels[s[i-k]] {
            count--
        }
        if count > maxCount {
            maxCount = count
        }
    }
    return maxCount
}
```

### Implementation in Java

```java
public class Solution {
    public int maxVowels(String s, int k) {
        Set<Character> vowels = Set.of('a', 'e', 'i', 'o', 'u');
        int count = 0;
        for (int i = 0; i < k; i++) {
            if (vowels.contains(s.charAt(i))) {
                count++;
            }
        }
        int maxCount = count;
        for (int i = k; i < s.length(); i++) {
            if (vowels.contains(s.charAt(i))) {
                count++;
            }
            if (vowels.contains(s.charAt(i - k))) {
                count--;
            }
            maxCount = Math.max(maxCount, count);
        }
        return maxCount;
    }
}
```

### Implementation in Python
```python
class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        vowels = {'a', 'e', 'i', 'o', 'u'}
        count = sum(1 for i in range(k) if s[i] in vowels)
        max_count = count

        for i in range(k, len(s)):
            if s[i] in vowels:
                count += 1
            if s[i - k] in vowels:
                count -= 1
            max_count = max(max_count, count)

        return max_count
```
