## Description
This **Password Manager CLI** is a terminal-based password management tool built using **Rust** with the **Ratatui** library for an interactive user interface. It allows users to securely store and manage their account credentials.

## Features

- 🔑 **Securely store passwords**
- 🖥️ **TUI-based interactive interface** using `ratatui`
- 📄 **Save & load credentials** from a text file
- ⬆️⬇️ **Navigation** using arrow keys
- 📝 **Add new accounts & passwords**
- 🔍 **View stored passwords**
- ❌ **Exit easily with 'q' key**

## Technologies Used

- 🦀 **Rust** - Safe and fast systems programming language
- 📦 **Crossterm** - Terminal handling library
- 🎨 **Ratatui** - TUI framework for interactive CLI applications



## Installation & Usage
### Clone the Repository
```bash
git clone https://github.com/MamangRust/example-password-manager-ratatui.git
cd example-password-manager-ratatui
```

### Build and Run
```bash
cargo run
```
