## Description

**Subnet Calculation Ratatui** is a terminal-based application for calculating subnet information such as network address, broadcast address, subnet count, and host count. It is built using [Ratatui](https://github.com/ratatui-org/ratatui), a Rust library for creating rich terminal user interfaces.


## Features
- 🔢 **Interactive Input:** Input an IP address and subnet mask interactively.
- 🌐 **Network Calculations:** Calculate network address and broadcast address.
- 🧩 **Subnet Analysis:** Determine the number of available subnets and hosts per subnet.
- 📊 **Structured Display:** Display results in a structured terminal UI.

## Technologies Used

- 🦀 **Rust** - Safe and fast systems programming language
- 📦 **Crossterm** - Terminal handling library
- 🎨 **Ratatui** - TUI framework for interactive CLI applications
