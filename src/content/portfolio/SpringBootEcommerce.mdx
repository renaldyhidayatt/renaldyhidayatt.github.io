# Spring Boot E-Commerce API

**High-Performance Backend API for a Full-Featured E-Commerce Platform**

The `Spring Boot E-Commerce API` is a high-performance backend system designed to support a complete digital commerce ecosystem. Built with **Java 21** and **Spring Boot**, it provides a comprehensive set of capabilities including JWT-based authentication, user and role management, product and category management, shopping cart operations, order processing, product reviews, merchant management, and shipping address handling.

---

## Core Features

### 1. Authentication & Security

* **JWT Authentication:** Secure user registration and login using JSON Web Tokens.
* **Role-Based Access Control (RBAC):** Granular permission model separating access for administrators, merchants, and customers.

### 2. Product & Category Management

* **Product CRUD Operations:** Full lifecycle management for product entities, including creation, updates, retrieval, and deletion.
* **Category Management:** Structured product categorization to improve searchability and navigation across the platform.

### 3. Shopping Cart & Order Management

* **Shopping Cart System:** Real-time cart operations including add, update quantity, remove items, and cart retrieval.
* **Order Processing:** End-to-end order lifecycle management with total price calculation and transactional consistency.

### 4. Merchant & Business Features

* **Merchant Onboarding:** Enables merchants to register and operate within the platform.
* **Merchant Profiles:** Supports detailed merchant identity including branding and business metadata.

### 5. Reviews & Shipping

* **Product Reviews:** Users can submit ratings, comments, and media attachments for purchased products.
* **Shipping Address Management:** Structured address handling to support checkout and delivery workflows.

---

## Technology Stack

* **Programming Language:** Java 21
* **Framework:** Spring Boot 3.x
* **Security:** Spring Security with JWT
* **ORM / Data Layer:** Spring Data JPA (Hibernate)
* **Database:** PostgreSQL
* **Build Tool:** Maven
* **API Documentation:** Springdoc OpenAPI (Swagger UI)

## Getting Started

### Prerequisites

* Java Development Kit (JDK) 21 or higher
* Apache Maven
* PostgreSQL (running locally or remotely)

### Installation Steps

**1. Clone the repository**

```bash
git clone https://github.com/MamangRust/example-ecommerce-springboot-new
cd example-ecommerce-springboot-new
```

**2. Configure the database**
Update `src/main/resources/application.properties`:

```properties
spring.datasource.url=jdbc:postgresql://localhost:5432/ecommerce_db
spring.datasource.username=postgres
spring.datasource.password=postgres
spring.jpa.hibernate.ddl-auto=update
```

**3. Run the application**

```bash
./mvnw spring-boot:run
```

**4. Access the API**

* Base URL: `http://localhost:8080`
* Swagger UI: `http://localhost:8080/swagger-ui.html`
