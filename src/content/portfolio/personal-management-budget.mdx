## Project Description

The **Personal Financial Management App** is a personal finance management application designed to help users manage their finances more effectively. This app allows users to track transactions, set budgets, and establish financial goals.

The app is built using **Tauri** to create a lightweight desktop application, with **React** for the frontend and **SQLite** as the main database.

## Technologies Used


- 🖥️ **Tauri** - Wraps the app into a cross-platform desktop application.
- ⚛️ **React** - A frontend framework for building interactive user interfaces.
- 🗄️ **SQLite** - A local database to store user and transaction data.
- 🎨 **Ant Design** - A UI library for building a modern, responsive app interface.

## Key Features

### 🔐 User Authentication
- Users can register, log in, and manage their accounts securely.

### 💰 Transaction Management
- Track income, expenses, and transfers.
- Categorize transactions for better organization.
- View detailed transaction history.

### 📊 Budgeting
- Set budgets for different expense categories.
- Monitor budget usage and receive alerts for overspending.

### 🎯 Goal Setting
- Create financial goals with target amounts and deadlines.
- Track progress toward achieving financial goals.

## How to Run the App

### Prerequisites
- **Node.js** and **npm** must be installed.
- **Rust** and **Cargo** must be installed.
- **Tauri CLI** installed globally.


### Steps

1. **Clone the Repository**

   ```bash
   git clone https://github.com/MamangRust/personal-financial-management.git
   cd personal-financial-app

2. **Install Depencies**
   ```bash
    yarn install
   ```
3. **Install Depencies src-tauri**
    ```bash
    cd src-tauri

    cargo build

    ```

4. **Build tauri App**
    ```bash
    yarn tauri build

    running on release folder src-tauri
    ```
