# 🚀 Pomodoro Task QT

**Pomodoro Task QT** is a Pomodoro Timer application with task management features designed to enhance productivity.

## ✨ Key Features

✅ **Tasks List & Pomodoro Timer** - Organize tasks and use the Pomodoro technique for focused work.

📊 **Monthly Visualization** - Analyze monthly productivity in graphical form.

📆 **Weekly Visualization** - View weekly statistics to understand work patterns.
