# 📦 Inventory Django

This project is built using **Django** as the main framework. Django was chosen for its simplicity and comprehensive features for database management and routing.

## ✨ Features

- CRUD operations with **Django ORM** 🗂️
- **User authentication** 🔐
- **Interactive dashboard** 📊

## 🛠 Technologies Used

- **Django** 🦄
- **PostgreSQL** 🗃️
- **Bootstrap** ⚙️
