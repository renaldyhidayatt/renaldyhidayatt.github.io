## Project Description

The **Movie GRPC** project is a **CRUD** application for managing movies, built with **Golang and GORM**. The application uses **gRPC** for communication between services and is supported by observability tools including **Jaeger, OpenTelemetry (Otel), and Prometheus** for monitoring and tracing. All services run in a **Docker** environment for easy deployment.
