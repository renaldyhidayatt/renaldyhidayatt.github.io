# **Welcome to the Payment Gateway Project!**

This system simulates a payment gateway handling operations such as:

- Card Transactions 💳
- Top-ups 💰
- Withdrawals 🏧
- Transfers 🔄

The project involves multiple entities like users, cards, merchants, and financial transactions.

## **Tech Stack** 🚀

### Backend

- **Golang** 🐹
- **Sqlc** 🗄️
- **Echo Framework** ⚡
- **gRPC** 🔌

### Frontend

- **ReactJs** ⚛️
- **Shadcn UI** 🎨
- **Tailwind.css** 🌊
- **Rust** 🦀
- **Tauri.js** 🚀
- **Axios** 🔗
- **Zustand** 🐻

## **Get Started** 🏁

To clone and run the project:

### Backend

```sh
git clone https://github.com/MamangRust/payment-gateway-grpc.git
cd payment-gateway

go run cmd/migrate/main.go // migrate table
go run cmd/client/main.go // run client (rest api)
go run cmd/server/main.go // run server (grpc)

```

### Frontend-web
```sh
git clone https://github.com/MamangRust/payment-gateway-web.git

cd payment-gateway-web

npm install

npm run dev

```


### Frontend-dashboard
```sh
git clone https://github.com/MamangRust/payment-gateway-dashboard.git

cd payment-gateway-web

npm install

// build tauri app
npm run tauri build

# Hasil build ada di:
ls src-tauri/target/release
```
