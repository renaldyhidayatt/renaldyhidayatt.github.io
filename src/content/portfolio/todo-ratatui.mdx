## Description
Example Todo Ratatui is a terminal-based todo application built using Rust and the Ratatui library. It includes Pomodoro timer functionality to help users manage tasks efficiently.

## Features
- **Task Management**: Add, delete, and navigate tasks with keyboard shortcuts.
- **Pomodoro Timer**: Integrated Pomodoro timer for productivity.
- **Interactive UI**: Uses Ratatui for an easy-to-read and user-friendly interface.
- **Simple Navigation**: Press `q` to exit the application.

## Technologies Used
- 🦀 **Rust** - Main programming language
- 🎨 **Ratatui** - Library for creating terminal-based UI
- ⌨️ **Crossterm** - Library for handling keyboard input and terminal display
