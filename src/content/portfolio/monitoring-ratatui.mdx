## Description
Example System Monitoring Ratatui is a terminal-based system monitoring application built using Rust and the Ratatui library. It provides real-time information on CPU usage, RAM consumption, and network activity in an interactive display.

## Features
- **CPU Monitoring** 🧑‍💻: Displays CPU usage with gauges and bar charts.
- **RAM Monitoring** 🧠: Displays RAM usage with gauges and bar charts.
- **Network Monitoring** 🌐: Displays received and transmitted data in KB.
- **Interactive UI** 🎮: Uses **Ratatui** for an easy-to-read and user-friendly interface.
- **Simple Navigation** 🔲: Exit the application by pressing `q`.


## Technologies Used
- 🦀 **Rust** - Main programming language
- 🎨 **Ratatui** - Library for creating terminal-based UI
- ⌨️ **Crossterm** - Library for handling keyboard input and terminal display
- 🖥️ **Sysinfo** - Library for retrieving system information
