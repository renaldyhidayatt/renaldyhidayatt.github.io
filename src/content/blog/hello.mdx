## Hello World in Python

Python is a versatile and beginner-friendly programming language. Let's start with the traditional "Hello World" program to get a feel for how Python works.

## Writing Your First Python Program

To write a "Hello World" program in Python, follow these steps:

1. **Open a Text Editor:** You can use any text editor or code editor of your choice. Create a new file and save it with a `.py` extension, like `hello.py`.

2. **Write the Code:** In your `.py` file, enter the following code:

```python
print("Hello, World!")
```
